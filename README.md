# WinterHackathon Arcade

Короткий питч: это зимний мини-парк с четырьмя аркадами. Играешь, зарабатываешь внутриигровые монеты и можешь закинуть их в наш «казик», чтобы претендовать на приз, который готовят админы.

## Как это работает
- выбираешь одну из мини-игр и запускаешь сессию;
- набираешь очки, превращая их в монеты;
- решаешь: копить дальше или рискнуть в казино ради приза от админов.

## Реализованные игры
- Flappy Gift — зимняя вариация классического flappy.
- osu! lite — быстрые тапы в ритм.
- Ski Run — слалом по снежной трассе.
- Ice Lumberjack — успей порубить лёд, пока не упадёт.
